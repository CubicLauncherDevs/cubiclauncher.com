# Instalación de CubicLauncher

Para comenzar a usar CubicLauncher, asegúrate de cumplir con los requisitos mínimos y seguir los pasos de instalación para tu sistema operativo.

## Requisitos previos

CubicLauncher es ahora una aplicación nativa, lo que significa que no requiere que tengas Java instalado para *ejecutar el launcher mismo*. Sin embargo, **Minecraft** sigue necesitando Java para funcionar.

- **Para el Launcher**: No requiere Java (es nativo).
- **Para Minecraft**: Se recomienda **OpenJDK 21** (Adoptium Temurin) para las versiones más recientes del juego.
- **Memoria**: Al menos 2GB de RAM recomendados para el sistema.
- **Espacio**: El launcher es muy ligero (~10-20MB), pero Minecraft requerirá espacio adicional para versiones, assets y mundos.

## Métodos de Instalación

CubicLauncher se distribuye en formatos nativos para cada plataforma, ofreciendo una mejor integración y rendimiento.

### 1. Instaladores Nativos (Recomendado)
- **Windows**: Descarga el archivo `.exe` e instálalo como cualquier aplicación.
- **Linux**: Proporcionamos paquetes `.deb` (Debian/Ubuntu) y `.AppImage` para compatibilidad universal.
- **macOS**: Descarga el archivo `.dmg`, ábrelo y arrastra CubicLauncher a tu carpeta de Aplicaciones.

### 2. Archivo Universal (.JAR) - Legacy
Aunque el launcher principal es ahora nativo, mantenemos soporte para la versión legacy en Java para sistemas específicos.
1. **Descarga**: Obtén el archivo `CubicLauncher.jar`.
2. **Ejecución**: Requiere Java 21 instalado.
   - `java -jar CubicLauncher.jar`

Para una guía detallada paso a paso sobre cómo configurar Java para Minecraft, consulta nuestra [Sección de Guías](#guias/instalacion).
